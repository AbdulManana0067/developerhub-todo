# developerhub-todo
A simple Flutter to‑do list app with SharedPreferences persistence and a bottom-sheet quick add.

DeveloperHub To‑Do — a lightweight Flutter to‑do list application that demonstrates task creation via a bottom sheet, task display in a ListView, and local persistence using SharedPreferences. Ideal as a starter project or demo for learning state management and local storage in Flutter.


- Add tasks via a quick-add bottom sheet
- Display tasks in a ListView with completion toggle and delete
- Persist tasks locally using SharedPreferences (JSON)
- Simple, themeable Material UI
